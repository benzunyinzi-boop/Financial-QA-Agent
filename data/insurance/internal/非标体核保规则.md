# 非标体核保规则

## 一、什么是非标体
"标准体"指身体健康、无既往病史、无高危因素的投保人。"非标体"指存在以下任一情况：
- 既往疾病或体检异常
- 高危职业、高危爱好
- 超重 / 吸烟 / 酗酒等生活方式风险
- 家族遗传病史

## 二、核保结论分类

### 1. 标准体承保（SC）
按标准费率正常承保，无任何附加条件。

### 2. 加费承保（PC）
按一定比例上浮保费，常见加费幅度：
- 25% / 50% / 75% / 100% / 150% / 200%

### 3. 责任除外（EX）
除外特定疾病或部位，例如：
- "甲状腺及其相关疾病除外"
- "右乳及右乳相关疾病除外"

### 4. 延期承保（PP）
6 个月或 1 年后再投保，常用于：
- 近期手术 / 急性疾病恢复期
- 体检异常需要随访观察
- 妊娠期

### 5. 拒保（DC）
当前风险无法接受，常见拒保情形：
- 已确诊的恶性肿瘤、心脑血管严重疾病
- 严重精神类疾病
- 高度风险职业且无法解除

## 三、常见非标体核保参考（示例）

### 1. 甲状腺结节
| 情况 | 重疾险 | 医疗险 |
|---|---|---|
| TI-RADS 1-2 级 | 标体 | 标体 |
| TI-RADS 3 级 | 标体 / 甲状腺除外 | 甲状腺除外 |
| TI-RADS 4A 及以上 | 拒保（建议手术后再申请） | 拒保 |
| 已手术（良性，病理明确） | 加费 25%-50% / 除外 | 甲状腺除外 |
| 已手术（恶性） | 拒保 | 拒保 |

### 2. 乳腺结节
| 情况 | 重疾险 | 医疗险 |
|---|---|---|
| BI-RADS 1-2 级 | 标体 | 标体 |
| BI-RADS 3 级 | 标体 / 乳腺除外 | 乳腺除外 |
| BI-RADS 4 级 | 拒保 | 拒保 |
| 已手术（良性） | 加费 / 除外 | 乳腺除外 |

### 3. 高血压
| 情况 | 重疾险 | 医疗险 |
|---|---|---|
| 1 级（140-159 / 90-99） | 加费 25%-50% | 高血压及并发症除外 |
| 2 级（160-179 / 100-109） | 拒保 | 拒保 |
| 3 级及合并并发症 | 拒保 | 拒保 |

### 4. 糖尿病
| 情况 | 重疾险 | 医疗险 |
|---|---|---|
| 妊娠糖尿病（已恢复） | 标体 | 妊娠相关除外 |
| 2 型糖尿病（无并发症） | 多数拒保 | 拒保 |
| 1 型糖尿病 | 拒保 | 拒保 |

### 5. BMI 异常
| BMI 范围 | 核保结论 |
|---|---|
| 18.5 - 28 | 标体 |
| 28 - 32 | 加费 25%-50% |
| 32 - 35 | 加费 50%-100% |
| > 35 | 拒保 |

## 四、人工核保流程
当智能核保无明确结论时，启动人工核保：

1. **资料收集**
   - 病历资料（门诊病历、住院病历、出院小结）
   - 检查报告（病理、影像、化验）
   - 治疗经过
   - 当前状态

2. **资料审核**（核保员）
   - 审核病史完整性
   - 评估当前风险等级
   - 参考行业标准（再保公司核保手册）

3. **结论给出**
   - 标体 / 加费 / 除外 / 延期 / 拒保

4. **客户确认**
   - 客服将结论告知客户
   - 客户接受后继续投保流程
   - 客户拒绝可申请其他产品或终止申请

## 五、特殊情形处理

### 多家公司被拒保
若客户告知曾被其他公司拒保，需：
- 询问拒保原因
- 提供拒保通知书
- 重新评估当前风险
- 不可隐瞒拒保历史

### 健康告知争议
客户对某项健康告知存疑时：
- 客服不得自行判断"填否还是填是"
- 引导客户如实告知，由人工核保审核
- 留痕：录音保存客户原始描述
